# docs

Put the app screenshot here as **`screenshot.png`** — the main README embeds
`docs/screenshot.png` at the top.

To take a good one:

1. Start the app: `./run.ps1` → open <http://localhost:8000>.
2. Click the three suggested prompts (or type a few messages) so the graph
   fills with nodes.
3. Capture the window (Windows: `Win + Shift + S`), save it here as
   `screenshot.png`.
